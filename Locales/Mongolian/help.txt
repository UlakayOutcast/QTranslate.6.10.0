Системийн шууд товчлуурууд (defaults):
Дабил Ctrl => Програмын үндсэн цонхыг харуулах (хэрэв орчуулсан текст байгаа бол)
Ctrl + Q => Сонгосон текстийг орчуулж цонхонд харуулах
Ctrl + Shift + Q => Толь бичгийн цонхыг харуулах
Ctrl + E => Сонгосон текстийг сонсох

Үндсэн цонхны шууд товчлуурууд:
Ctrl + Enter => Текстийг орчуулах
Ctrl + N => Одоогийн орчуулгыг цэвэрлэх
Ctrl + D => Толь бичиг харуулах
Ctrl + Alt + 1..9 => Сонгогдсон (1..9 дэх) үйлчилгээтэй толь бичгийг харуулах
Ctrl + H => Орчуулгын түүхийг харуулах
Ctrl + K => Виртуал гарыг харуулах
Ctrl + Tab / Ctrl + Shift + Tab / Ctrl + 1..9 => Дараагийн / өмнөх / 1..9 дэх үйлчилгээ рүү шилжих
Shift + Esc => Хэлүүдийг анхдагч хос болгоно
Ctrl + Shift + 1..9 => Хэлний хослолыг сонгох
Ctrl + I => Орчуулгын чиглэлийг өөрчлөх
Ctrl + Space => Оруулсан үгийн сонголтуудын жагсаалт бүхий цэсийг харуулах (автоматаар бөглөх)
F1 => Тусламж үзүүлэх
F11 => Бүтэн дэлгэцийн горимыг идэвхжүүлэх / идэвхгүй болгох
Alt + Зүүн сум => Өмнөх орчуулга руу очих
Alt + Баруун сум => Дараагийн орчуулга руу очно уу
Ctrl + Up => Орчуулгыг текст оруулах цонхонд хуулах

Текстийг хулганаар сонгож орчуулах гурван горим байдаг (хэрхэн идэвхжүүлэх - доороос харна уу):
  1) Дүрс харуулах (дурын програмаас текст сонгоход -> QTranslate дүрс курсорын ойролцоо гарч ирнэ. Дүрсэн дээр дарж сонгосон текстийн орчуулгыг харах);
  2) Орчуулгыг харуулах (дурын програмаас текст сонгоход -> сонгосон текстийн орчуулгыг шууд харах);
  3) Орчуулгыг харуулах ба дуудлага хийх (дурын програмаас текстийг сонгоход -> сонгосон текстийн орчуулгыг харуулж TTS-р уншуулах).

Үйлдлүүд:
  - Мэдэгдлийн талбар дахь програмын дүрс дээр хулганаар нэг товшиход хулганаар сонгож орчуулах горимыг идэвхжүүлдэг (идэвхгүй болгодог);
  - Мэдэгдлийн талбар дахь програмын дүрс дээр хулганаар хоёр товшиход програмын үндсэн цонх нээгдэнэ;
  - Попап цонхны гарчгийг хоёр товшиход програмын үндсэн цонх одоогийн орчуулгаар нээнэ;
  - Үйлчилгээний нэр дээр хулганы товчийг дарахад тухайн үйлчилгээ өөрчлөгдөж, текстийг дахин хөрвүүлдэг;
  - Үйлчилгээний нэр дээрх хулганы дунд товчийг дарахад үйлчилгээ хөтөч дээр нээгдэнэ;
  - Үйлчилгээний нэр дээр хулганы баруун товчийг дарахад сонгосон үйлчилгээнд үйлчилгээ нэмэлтээр орох болно.
Баярлалаа.
